# Noma AI

A smart farming insights application designed for farmers in Bali, Nigeria. Noma (meaning "farming" in Hausa) combines weather data with AI to provide intelligent agricultural recommendations.

## Features

- Real-time weather monitoring
- Agricultural activity recommendations
- Dark mode support
- Responsive design
- Activity-based filtering
- Weather metric visualization

## Tech Stack

- React 18
- Tailwind CSS
- React Query
- Lucide React Icons
- Modern UI Components

## Local Development

1. Clone the repository
2. Install dependencies:
   ```bash
   cd client
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser

## Features

### Weather Metrics
- Temperature
- Humidity
- Wind Speed & Direction
- UV Index
- Pressure
- Cloud Cover

### Agricultural Activities
- Planting
- Harvesting
- Irrigation
- Fertilizing
- Pest Control

### Smart Recommendations
The app provides intelligent recommendations based on:
- Current weather conditions
- Selected agricultural activity
- Historical weather patterns
- Best farming practices
